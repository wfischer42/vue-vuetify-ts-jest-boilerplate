# Boilerplate for Vue project using

- Vue
- Vuetify
- Typescript
- Preprocessors
  _ Stylus
  _ Pug
- Storybook
- Storybook Snapshots
- Jest

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Run your unit tests

```
npm run test:unit
```

### Run your tests & update snapshots

```
npm run test:unit -- -u
```

### Lints and fixes files

```
npm run lint
```
